# Analisis-Sentimen  
Proyek Analisis Sentimen terhadap Data Teks

## Deskripsi Proyek  
Proyek ini bertujuan untuk melakukan analisis sentimen pada data teks menggunakan metode machine learning dan text processing. Notebook mencakup tahapan preprocessing, analisis teks, pelatihan model, dan evaluasi performa. Seluruh eksperimen dijalankan di lingkungan **Jupyter Notebook** atau **Google Colab**.

## Fitur & Alur Kerja  

### Pemrosesan & Analisis Teks  
- Memuat dataset ke notebook  
- Pra-proses teks (cleaning, tokenizing, normalisasi)  
- Ekstraksi fitur teks  

### Pelatihan Model & Klasifikasi Sentimen  
- Melatih model untuk mengklasifikasi sentimen  
- Penanganan ketidakseimbangan kelas jika diperlukan  
- Evaluasi menggunakan akurasi, precision, recall, dan F1-score  

### Visualisasi & Interpretasi  
- Visualisasi distribusi sentimen  
- Analisis hasil model dan error yang terjadi  

## Teknologi & Dependencies  
Notebook menggunakan library Python berikut:  
- pandas, NumPy  
- scikit-learn  
- nltk / Sastrawi (opsional, sesuai implementasi)  
- Matplotlib / Seaborn  
- Jupyter Notebook / Google Colab  

## Cara Menjalankan  
1. Buka file notebook di Jupyter Notebook atau Google Colab  
2. Muat dataset sesuai instruksi pada notebook  
3. Jalankan seluruh sel secara berurutan  
4. Amati hasil preprocessing, pelatihan, dan evaluasi model
